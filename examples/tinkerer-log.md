# How to Predict-RLM in a hurry

First start w claude code once you've installed predict-rlm's skill

```claude
❯ /rlm help me build an RLM that uses https://agent-browser.dev/ (already installed) that will take an event page url, crawl it and its subpages
  up to depth N. and create me a markdown executive summary of who's gonna be there, who's going to present. the RLM should also have a perplexity
  search api tool that allows it to refine information about who's who, whats what, etc.
```
